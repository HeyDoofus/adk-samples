# Project Brief

## Core Requirements & Goals

- Provide a collection of ready-to-use sample agents built on the Google Agent Development Kit (ADK).
- Accelerate development by offering functional starting points for common agent use cases.
- Showcase various agent capabilities, from simple bots to complex workflows.
- Enable users to easily explore, set up, run, and customize these sample agents.
